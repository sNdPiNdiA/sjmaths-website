Basic Distance Formula

$d = \sqrt{(x_2 - x_1)^2 + (y_2 - y_1)^2}$

1. Find the distance between the following pairs of points:

(i) (3, 8), (6, 4)

Substitution: $d = \sqrt{(6-3)^2 + (4-8)^2}$
Calculation: $d = \sqrt{3^2 + (-4)^2} = \sqrt{9 + 16} = \sqrt{25}$
Result: 5 units

(ii) (-6, 8), (-2, 4)

Substitution: $d = \sqrt{(-2 - (-6))^2 + (4 - 8)^2}$
Simplify: $d = \sqrt{(4)^2 + (-4)^2} = \sqrt{16 + 16} = \sqrt{32}$
Result: $4\sqrt{2}$ units

(iii) (x, y), (-x, -y)

Substitution: $d = \sqrt{(-x - x)^2 + (-y - y)^2}$
Simplify: $d = \sqrt{(-2x)^2 + (-2y)^2} = \sqrt{4x^2 + 4y^2}$
Factor out 4: $d = \sqrt{4(x^2 + y^2)}$
Result: $2\sqrt{x^2 + y^2}$ units

Geometric Shapes & Collinearity

2. Find the distance between the points (0, 0) and (12, 5).

Substitution: $d = \sqrt{(12-0)^2 + (5-0)^2} = \sqrt{12^2 + 5^2}$
Calculation: $d = \sqrt{144 + 25} = \sqrt{169}$
Result: 13 units



3. Determine if the points (1, 4), (3, 2) and (-3, 10) are collinear.

Let A(1, 4), B(3, 2), C(-3, 10).
Distance AB: $\sqrt{(3-1)^2 + (2-4)^2} = \sqrt{4 + 4} = \sqrt{8} = 2\sqrt{2}$
Distance BC: $\sqrt{(-3-3)^2 + (10-2)^2} = \sqrt{36 + 64} = \sqrt{100} = 10$
Distance AC: $\sqrt{(-3-1)^2 + (10-4)^2} = \sqrt{16 + 36} = \sqrt{52} = 2\sqrt{13}$
Check: $AB + AC \neq BC$ ($2\sqrt{2} + 2\sqrt{13} \neq 10$).
Result: No, they are not collinear.



4. Check whether (2, 2), (5, 6) and (8, 2) are the vertices of an isosceles triangle.

Let A(2, 2), B(5, 6), C(8, 2).
AB: $\sqrt{(5-2)^2 + (6-2)^2} = \sqrt{9 + 16} = \sqrt{25} = 5$
BC: $\sqrt{(8-5)^2 + (2-6)^2} = \sqrt{9 + 16} = \sqrt{25} = 5$
AC: $\sqrt{(8-2)^2 + (2-2)^2} = \sqrt{36 + 0} = 6$
Since $AB = BC = 5$.
Result: Yes, it is an isosceles triangle.



5. In a hall, 4 students are seated at points A(1, 1), B(4, 4), C(7, 1) and D(4, -2). Ram asks Shyam, "Do you think ABCD is a square?" Shyam disagrees. Using distance formula, find who is correct.

Sides:
$AB = \sqrt{(4-1)^2 + (4-1)^2} = \sqrt{18}$
$BC = \sqrt{(7-4)^2 + (1-4)^2} = \sqrt{18}$
$CD = \sqrt{(4-7)^2 + (-2-1)^2} = \sqrt{18}$
$DA = \sqrt{(1-4)^2 + (1-(-2))^2} = \sqrt{18}$
All sides are equal. Now check diagonals.
Diagonals:
$AC = \sqrt{(7-1)^2 + (1-1)^2} = \sqrt{36} = 6$
$BD = \sqrt{(4-4)^2 + (-2-4)^2} = \sqrt{36} = 6$
Diagonals are equal.
Result: ABCD is a Square. Ram is correct.

Finding Unknowns

7. Find the point on the x-axis which is equidistant from (3, -4) and (-5, 6).

Let point be P(x, 0). Given A(3, -4) and B(-5, 6).
Condition: $PA^2 = PB^2$.
$(x-3)^2 + (0-(-4))^2 = (x-(-5))^2 + (0-6)^2$
$x^2 - 6x + 9 + 16 = x^2 + 10x + 25 + 36$
$-6x + 25 = 10x + 61$
$-16x = 36 \Rightarrow x = -36/16 = -9/4$
Result: Point is (-2.25, 0)



8. Find the values of $y$ for which the distance between the points P(3, -2) and Q(9, y) is 10 units.

Distance formula: $PQ^2 = 100$.
$(9-3)^2 + (y-(-2))^2 = 100$
$36 + (y+2)^2 = 100 \Rightarrow (y+2)^2 = 64$
$y+2 = \pm 8$
Case 1: $y+2 = 8 \Rightarrow y=6$
Case 2: $y+2 = -8 \Rightarrow y=-10$
Result: y = 6 or -10



9. If Q(0, 1) is equidistant from P(6, -4) and R(x, 7), find the values of x. Also find the distances QR and PR.

Find x: $QP^2 = QR^2$
$(6-0)^2 + (-4-1)^2 = (x-0)^2 + (7-1)^2$
$36 + 25 = x^2 + 36 \Rightarrow x^2 = 25 \Rightarrow x = \pm 5$
Distance QR (using x=5): $QR = \sqrt{61}$
Distance PR (Case 1: x=5): $PR = \sqrt{(5-6)^2 + (7-(-4))^2} = \sqrt{1+121} = \sqrt{122}$
Distance PR (Case 2: x=-5): $PR = \sqrt{(-5-6)^2 + (7-(-4))^2} = \sqrt{121+121} = 11\sqrt{2}$
Result: x = Â±5; QR = $\sqrt{61}$; PR = $\sqrt{122}$ or $11\sqrt{2}$
