Section E (4 Marks)

Q11. Case Study: Number Game

The sum of the digits of a two-digit number is 9. Also, nine times this number is twice the number obtained by reversing the order of the digits.

(i) Form the pair of linear equations representing the situation.

(ii) Find the number.

(iii) If 27 is added to the number, what is the new number obtained?

Solve on white paper.
